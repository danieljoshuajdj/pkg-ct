# Support

Thanks for using `@danijsrr/pkg-ct`.

## Where to get help

- Read the [README](./README.md) for the fastest overview.
- Use the [command reference](./docs/commands.md) for CLI options.
- Check the [FAQ](./docs/faq.md) for common false positives and setup issues.
- Open a GitHub issue for reproducible bugs or documentation gaps.

## Good bug reports include

- the command you ran;
- your Node.js version;
- your package manager and lockfile type;
- a small reproduction, fixture, or sanitized output;
- whether online metadata was available.

## Security reports

Please follow [SECURITY.md](./SECURITY.md) instead of opening a public issue.
